Movie Recommendation System

Features : Genre, Rating, Year

Process:
Data set -> Feature Engineering -> One-Hot Encoding -> Feature Scaling -> Cosine Similarity -> Ranking -> Top-N Recommendations

I started learning it by building a basic movie recommendation system where the movies with same genre are recommended.
Later by applying the Machine Learning concepts such as feature engineering,scaling,cosine similairty,etc, 
I learned how they would affect the recommendation quality and implemented them in this system.
